# OpenJTalk 先頭記号除去パターン

`src/voicevox_client.py` の `_LEADING_PUNCT_PATTERN` で除去される先頭文字の一覧。

## パターン定義

```python
_LEADING_PUNCT_PATTERN = re.compile(r"^[、。，．,.\s…ー－▶▼◆●○★☆※・►◀▲△▽▷◁→←↑↓]+")
```

## 実データで確認された例

`data/522d055285db/cleaned_text.txt` から抽出。

### 三角記号 `▶`

```
▶ビジネスへのエイキョウ。
▶ジュウギョウインへのエイキョウ。
▶シンライセイがサービスのテイキョウカチにチョッケツするサービスのバアイ。
▶ゲンジツテキなアプローチ。
▶クラウドプロバイダーテイキョウツール。
▶SaaSガタツール。
▶オープンソースソフトウェア。
▶ショウガイのエイキョウハンイがあいまい。
▶ポストモーテムをジッシするキジュンをキメル。
```

### 中黒 `・`

```
・コンプリケイテッド・サブシステムチーム。
・コラボレーション。
・メリット。
・デメリット。
```

### 読点 `、`

```
、ニュウモンカンシイチモダンなモニタリングのための、デザインパターン、は...
、システムウンヨウアンチパターンーエンジニアがデブオプスでカイケツする...
```

### 長音記号 `ー`

```
ートのミノガシやテキセツなタイオウのケツジョがモンダイとして...
```

## パターン一覧

### 実データで確認済み

| 文字 | 名称 | 出現 |
|------|------|------|
| `▶` | 黒右三角 | 多数 |
| `・` | 中黒 | あり |
| `、` | 読点 | あり |
| `ー` | 長音記号 | あり |

### 未確認（予防的に追加）

| 文字 | 名称 |
|------|------|
| `。` | 句点 |
| `，` | 全角カンマ |
| `．` | 全角ピリオド |
| `,` | 半角カンマ |
| `.` | 半角ピリオド |
| `\s` | 空白文字 |
| `…` | 三点リーダー |
| `－` | 全角ダッシュ |
| `▼►◀▲△▽▷◁` | 各種三角記号 |
| `→←↑↓` | 矢印 |
| `◆●○★☆※` | 装飾記号 |

## 除去される理由

OpenJTalk は先頭に句読点や記号があると以下の警告を発する:

```
WARNING: convert_pos() in njd2jpcommon.c: 記号 サ変接続 * * are not appropriate POS.
WARNING: JPCommonLabel_insert_pause() in jpcommon_label.c: First mora should not be short pause.
```
